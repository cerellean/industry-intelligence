# 行业情报收集 Skill (v1.0)

专为卫浴制造业及家居建材行业设计的智能情报收集与分析工具。基于用户的企业背景（中国卫浴头部企业），提供针对性的行业监测和竞争分析。

## 一、核心监测领域

### 🎯 1. 竞争对手监测
**重点监测品牌**：
- 国际品牌：科勒(Kohler)、TOTO、汉斯格雅(Hansgrohe)、高仪(Grohe)、美标(American Standard)
- 国内品牌：九牧、箭牌、恒洁、惠达、东鹏
- 新兴品牌：智能卫浴初创企业

**监测维度**：
- 新品发布（智能马桶、节水龙头、新材料应用）
- 价格策略调整
- 渠道扩张（线上/线下、国际市场）
- 营销活动（广告投放、促销活动）
- 技术专利申请

### 🔬 2. 技术趋势追踪
**关键技术方向**：
- 智能卫浴：AI识别、语音控制、健康监测
- 节水技术：超节水马桶、智能水龙头
- 新材料：抗菌陶瓷、自清洁涂层、环保材料
- 生产工艺：3D打印、智能制造、数字化工厂
- 设计趋势：极简主义、个性化定制、适老化设计

### 📜 3. 政策法规监测
**关键政策领域**：
- 环保标准：水效标识、VOC排放、废水处理
- 建筑规范：适老化卫生间标准、无障碍设计
- 进出口政策：关税调整、贸易壁垒、认证要求
- 产业政策：智能制造支持、绿色制造补贴
- 消费者保护：产品质量标准、售后服务规范

### 📊 4. 市场数据分析
**监测指标**：
- 市场规模（全球/中国/区域）
- 增长率（年增长率、季度变化）
- 市场份额（主要品牌市占率）
- 渠道结构（零售/工程/电商占比）
- 价格区间分布（高端/中端/低端）

### ⛓️ 5. 供应链情报
**监测重点**：
- 原材料价格：陶瓷原料、铜材、塑料、电子元件
- 供应商动态：主要供应商产能、技术升级
- 物流信息：海运费用、港口拥堵、运输时间
- 替代材料：新型替代材料的商业化进展

## 三、执行流程

### Step 1: 情报需求确认
每次执行前确认：
- 监测时间范围（默认：最近7天）
- 重点关注领域（默认：全部5个领域）
- 紧急程度（常规监测/深度分析）

### Step 2: 多源数据收集
使用 `web_search` 和 `web_fetch` 工具，按优先级分层收集：

**第一层：核心信息源（必须覆盖）**
1. 行业协会官网（政策、标准更新）
2. 主要竞品新闻中心（新品发布、战略调整）
3. 行业权威媒体（深度分析、趋势解读）

**第二层：专业信息源（重点覆盖）**
1. 垂直行业门户（产品评测、技术分析）
2. 研究机构报告（市场数据、预测分析）
3. 技术专利数据库（创新技术、专利布局）

**第三层：补充信息源（选择性覆盖）**
1. 社交媒体矩阵（行业动态、消费者反馈）
2. 相关领域媒体（技术融合、跨界创新）
3. 区域市场信息（本地化动态、区域政策）

**搜索策略**：
1. 按关键词矩阵分批搜索，每批≤50词
2. 优先使用site:限定搜索权威网站
3. 记录完整数据来源、时间戳和权威等级
4. 对重要信息进行交叉验证

### Step 3: 情报筛选与验证
筛选标准：
1. **相关性**：与卫浴行业直接相关
2. **时效性**：优先最新信息（24小时内最优）
3. **权威性**：官方来源 > 行业媒体 > 自媒体
4. **重要性**：重大战略调整 > 常规运营信息

### Step 4: 结构化分析
按以下框架组织信息：
```
1. 竞争动态
   - 新品发布
   - 价格调整
   - 渠道变化
   - 营销活动

2. 技术进展
   - 技术创新
   - 专利布局
   - 研发投入
   - 技术合作

3. 政策影响
   - 新规发布
   - 标准更新
   - 政策解读
   - 合规建议

4. 市场趋势
   - 规模变化
   - 结构演变
   - 区域差异
   - 消费偏好

5. 供应链情报
   - 价格波动
   - 供应稳定
   - 替代方案
   - 物流状况
```

### Step 5: 洞察生成与建议
基于分析结果：
1. **机会识别**：新兴市场、技术空白、政策红利
2. **风险预警**：竞争威胁、政策风险、供应链风险
3. **行动建议**：短期应对、中期布局、长期战略
