# 行业情报Skill使用示例

## 基础使用示例

### 示例1：常规周度监测
```
请使用industry-intelligence skill进行卫浴行业监测，时间范围：最近7天
```

**预期输出**：
- 一周行业动态汇总
- 主要展会信息
- 技术趋势分析
- 市场数据更新
- 战略建议

## 输出格式定制

### 简洁版报告
```
请生成简洁版行业情报简报，重点突出：关键动态、紧急事项、立即行动
```

### 详细版报告
```
请生成详细版行业情报报告，包含：数据支撑、案例分析、深度解读、多维度分析
```

### 管理层摘要
```
请生成为管理层准备的执行摘要，聚焦：战略影响、决策要点、资源需求、预期收益
```

### 技术团队报告
```
请生成为技术团队准备的专项报告，重点：技术参数、研发方向、专利分析、测试标准
```

## 频率设置建议

### 日常快报（每天上午）
```
请生成昨日行业快报，聚焦：突发事件、价格波动、政策发布
```

### 周度简报（每周一）
```
请生成上周行业简报，全面覆盖：展会、技术、市场、政策动态
```

### 月度分析（每月初）
```
请生成上月行业深度分析，包含：趋势总结、数据统计、预测分析
```

### 季度战略（每季度）
```
请生成本季度战略评估，包括：竞争态势、市场变化、战略调整建议
```

## 注意事项

1. **信息时效性**：明确指定时间范围，确保信息最新
2. **信息源选择**：根据需求选择合适的信息源组合
3. **输出深度**：根据受众调整报告详细程度
4. **行动导向**：确保报告包含可执行的建议
5. **持续优化**：根据使用反馈调整监测重点和输出格式
