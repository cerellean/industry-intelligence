# Industry Intelligence Skill

## 概述
行业情报收集与分析工具 - 专注于卫浴制造业及家居建材行业的竞争情报、技术趋势、政策监测和市场分析。为战略决策提供数据驱动的行业洞察。

## 功能特点
- 🎯 **竞争对手监测**：科勒、TOTO、九牧等主要品牌动态
- 🔬 **技术趋势追踪**：智能卫浴、节水技术、新材料应用
- 📜 **政策法规监测**：水效标识、环保标准、产业政策
- 📊 **市场数据分析**：市场规模、增长率、渠道结构
- 🌍 **全球视野**：覆盖中美欧日主要市场
- ⚡ **实时更新**：24小时内最新行业动态

## 安装说明
1. 将本skill文件夹复制到OpenClaw的skills目录
2. 确保文件结构完整：
   ```
   industry-intelligence/
   ├── SKILL.md          # 主技能文件
   ├── README.md         # 说明文档
   ├── references/       # 参考文件
   ├── scripts/          # 辅助脚本
   └── test/            # 测试文件
   ```

## 输出示例
skill将生成结构化的行业情报简报，包含：
- 执行摘要
- 竞争动态
- 技术趋势
- 政策法规
- 市场数据
- 战略洞察
- 行动建议

## 信息源配置
skill预配置了多层信息源：
1. **核心层**：行业协会、竞品官网、权威媒体
2. **扩展层**：国际设计媒体、科技媒体、研究机构
3. **补充层**：社交媒体、区域市场信息
